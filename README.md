# npp-explorer-plugin介绍
这是基于[Explorer]修改的[Notepad++]插件，只能用于x64版Notepad++。

目标只有一个：
- 供x64版Notepad++使用 

## 安装
把生成的**Explorer.dll**复制到Notepad++的插件目录中：

`C:\Program Files\Notepad++\plugins\`
